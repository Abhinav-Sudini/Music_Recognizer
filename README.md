## 🎶 How It Works
Upload a short audio clip (e.g., 10 seconds). The application processes the clip, generates its unique fingerprint,

and queries the MySQL database to find a matching song, returning the song's identity if found.

## 🛠️ Quick Setup
Clone the repository: git clone.

Navigate to the project: cd Music_Recognizer

Configure your MySQL database in settings.py.

Run migrations: python manage.py migrate

Load your song data and generate fingerprints.

Start the server: python manage.py runserver
